# Culura 🎬📚

Welcome to the official documentation for **Culura**!  
Culura is an Android application built with **Jetpack Compose** that displays a static list of movies and books for users to browse.

---

## 📱 Features

- 🎞️ **Movies List** – Browse a curated list of movies with title, description, and cover image.  
- 📚 **Books List** – Browse a curated list of books with title, author, and cover image.  
- 🌟 **Simple & Fast** – Lightweight app with static content for quick browsing.  
- 🖼️ **Jetpack Compose UI** – Modern, responsive UI built entirely with Jetpack Compose.  

---

## 🛠️ Technologies Used

- **Kotlin**  
- **Jetpack Compose** – For UI design  
- **Android Studio** – Development environment  
- **Material Design** – UI components
